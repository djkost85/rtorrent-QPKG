How do I use watch directory?

1. Browse to the watch dir

Open My computer and browse network drive with the Download share. Make sure Download share is already mapped as network drive in your PC (eg., Y:).

2. Create short link to watch directory

Drag and drop rtorrent's watch dir to favorites!

All done! :-)

Download any torrent file from your favorite site and save it using Save As dialog, but when the dialog appears, select watch directory from Favorites available on a left sidebar!
